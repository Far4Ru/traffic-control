# Управление перекрестком

## Описание

Интерактивная симуляция дорожного перекрестка с управлением через RAG-агента. Транспортные средства движутся по полосам, реагируют на светофоры и соблюдают дистанцию. Поведение всех объектов на сцене можно менять с помощью текстовых промптов на естественном языке.

## Технологии

- **PixiJS** — 2D рендеринг
- **TypeScript** — типизация
- **ECS (Entity-Component-System)** — архитектура
- **RAG (Retrieval-Augmented Generation)** — управление поведением через промпты

## Примеры промптов

- *"Сделай час пик с большим количеством машин"* — повышает плотность трафика
- *"Включи агрессивное вождение"* — увеличивает скорость и ускорение
- *"Отключи светофоры"* — переводит перекресток на правило правой руки
- *"Сделай свободное движение"* — снижает плотность, повышает скорость
